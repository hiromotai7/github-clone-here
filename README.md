# github-clone-here
Git clone here with hierarchy

## how to use

```
$ ./github-clone-here.sh git@github.com:hiromotai7/github-clone-here.git
```

```
$ ./github-clone-here.sh https://github.com/hiromotai7/github-clone-here.git
```

```
$ tree -L 2
.
└── hiromotai7
    └── github-clone-here.git
```
